# model-validation-methods-

## Описание
Проект посвящен методам валидации моделей машинного обучения, оптимизации гиперпараметров и отбору признаков. Включает практическую реализацию различных схем кросс-валидации, методов feature selection и оптимизации гиперпараметров на реальных данных.

## Ключевые компоненты
Валидация: K-Fold, Group K-Fold, Stratified K-Fold, Time Series Split
Оптимизация гиперпараметров: Grid Search, Random Search, Bayesian Optimization (Optuna)
Отбор признаков: Lasso, Permutation Importance, SHAP, корреляционный анализ
Разделение данных: случайное, по времени, на три части (train/val/test)

## Основные задачи
Реализация методов валидации
Сравнение схем кросс-валидации
Исследование методов отбора признаков
Оптимизация гиперпараметров моделей
Анализ утечек данных

## Технологии
Python, scikit-learn, Optuna, SHAP, pandas, NumPy
